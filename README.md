## 🧠 Core System Logic & Architecture
하이퍼 가디언의 심장부에는 세 가지 기술적 기둥이 존재합니다.

### 1. AI Intel Sorting Layer
시장의 수많은 ETF 중 어떤 것이 가장 유효한 전력인지 AI가 실시간으로 판단합니다.
* **Logic**: 최근 일주일간의 수익률 데이터를 전수 조사하여 각 전략 섹션 상단에 배치합니다.
* **Prompt Engineering**: "Financial Sovereign" 톤으로 조율된 Gemini API가 정렬의 근거를 보증합니다.

### 2. Behavioral Economic Risk Gauge
유저가 손실을 직면하게 만드는 행동경제학적 UI 요소입니다.
$$Loss Rate = \frac{Current Price - Purchase Price}{Purchase Price} \times 100$$
* **Gauge Logic**: 손실률이 -10%에 가까워질수록 게이지 바가 녹색 → 황색 → 적색으로 가변하며 심리적 경각심을 고취합니다.

### 3. Atomic Interactive Workflow
사용자의 예약 경험을 극대화하기 위해 팝오버(Popover) 기반의 3단계 상태 제어 시스템을 구축했습니다.
* **상태 A (미예약)**: '상장 예약 ∨' 버튼 노출.
* **상태 B (확정 액션)**: 팝오버 내 예약 클릭 시 데이터 즉시 저장 및 `st.rerun()`을 통한 물리적 창 닫기 수행.
* **상태 C (예약 완료)** : '🚨 구매 예약 완료 ∨' 버튼으로 즉시 전환 및 취소 프로토콜 가동.

우리는 프로젝트 운영 중 발생할 수 있는 보안 위협에 대응하여 시스템의 무결성을 한 단계 격상시켰습니다.

* **Environment Isolation**: `.gitignore` 및 GitHub Secrets를 활용하여 보안 민감 정보를 물리적으로 격리, 데이터 무결성을 보장합니다.

## 🚀 Future Roadmap: From BETA to FULL-AUTO
현재 하이퍼 가디언은 **BETA 버전**으로 운영되고 있으며, 정식 업데이트를 통해 진정한 자산 자동 수호 체계를 완성할 예정입니다.

1. **증권사 계좌 직접 연동**: 국내 주요 증권사 API와 통신하여 실시간 잔고 및 체결 데이터를 동기화합니다.
2. **0.1초 자동 매수**: 상장 일정에서 예약한 종목이 상장되는 즉시, 시장가로 자동 진입하여 기회비용을 최소화합니다.
3. **Full-Auto 방어 시스템**: 설정된 원칙(-10.0%)을 이탈하는 즉시, 시스템이 인간의 판단을 거치지 않고 **'빛의 속도로 자동 매도'**를 집행합니다.
